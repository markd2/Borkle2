# Borkle 2

Taking the learnings from [Borkle](https://github.com/markd2/borkle) and use
them to make a more flexible tool, as well as form an engine that can be
used in other applications and other platforms.

Most of the design and notes are handwritten in old-school notebooks.  If you
have questions or whatnot, feel free to drop me a line at markd@borkware.com

There are some decisions in the [Architecture Design Records](adr) directory.
